Features

Everything you need.
Nothing you don't.

🤖
AI-curated, not algorithmic

Gemini 2.5 Pro reads every article in full and selects the 20 most important, non-redundant stories. You set the editorial prompt — the AI writes for your interests.

🌍
9 native languages

Each edition is generated natively — not translated. The French edition writes in French from French sources. The Chinese edition thinks in Chinese. 1 language = 1 edition.

🔒
Self-hosted, yours

Your data never leaves your server. One SQLite file. One API key. Deploy to Fly.io in 5 minutes. No subscriptions, no lock-in, no algorithm deciding what you read.

📰
34+ trusted RSS sources

Automatically fills gaps from Reuters, BBC, FT, The Economist, NYT, Guardian, Nature, Wired — and language-native sources for each edition. Zero API keys.

🎯
Editorial prompt

Tell the AI who you are. It shapes every digest to your curiosity — not a generic algorithm's guess. Write once, benefit every morning.

🗞️
Per-story sources

Every story shows a "Read sources" modal listing the original articles with direct links. Full transparency — see exactly what the AI read to write each summary.

📱
PWA + Native ready

Install directly from Safari or Chrome. Or build a native iOS/Android app with Capacitor — one command, zero code changes. App ID: news.cupof.app.

💸
~$0.07 per digest

One OpenRouter API key. One structured call. 20 stories summarised at once. At Gemini 2.5 Pro rates, a Starbucks latte buys you three months of daily digests.

How it works

Pipeline fires at 6 AM.
You wake up to a briefing.

01
You submit links

Drop URLs via the admin panel, API, or Apple Shortcuts / bookmarklet. Articles you saved all week become inputs for your digest. No links? The RSS fallback covers it.

02
Pipeline fires at 6 AM

At 6 AM GMT, the pipeline fires. Jina Reader extracts full text from every link — including paywalled articles, YouTube transcripts, TikTok captions. No API key needed. Then it fetches 34+ trusted RSS sources to fill any gaps.

03
AI curates 20 stories

One structured API call to Gemini 2.5 Pro selects the 20 best stories: no duplicates, mandatory geographic and category diversity, closing quote included. Written natively in the edition's language.

04
You review and publish

The digest lands in a draft state. Admin panel lets you edit, swap, or remove any story. One click to publish. Or set AUTO_PUBLISH=true for fully hands-off delivery.

05
Read in any language

Switch editions with the flag selector. Each language edition generates independently and is available simultaneously. Use 9 languages — one for each region or audience.

Native App

App Store ready.
Zero extra code.

🍎
Capacitor (iOS)

Run npx cap add ios && npx cap sync and open in Xcode. The same React codebase wraps into a real native shell. Submit to the App Store or sideload via TestFlight.

🤖
Capacitor (Android)

Run npx cap add android && npx cap sync and open in Android Studio. Sign, build, upload to the Play Store. Same code, no rewrites.

🌐
PWA (no store)

Visit app.cupof.news in Safari → Share → Add to Home Screen. Full-screen, icon on the home screen, works offline for cached content. Available today on any device — no App Store needed.

☁️
Median / AppMySite

Point Median at https://app.cupof.news. It wraps the URL into a native iOS + Android app, handles App Store submission, and updates automatically when you deploy. Zero code changes.
